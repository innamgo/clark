# Mermaid Markdown Viewer

A simple, beautifully designed web application that allows you to upload Markdown (`.md`) files and instantly view their formatted text and Mermaid diagrams.

## Features
- **Instant Rendering**: Uses `marked.js` to render markdown and `mermaid.js` for diagram rendering.
- **Local Processing**: All parsing and rendering happens directly in your browser. No server required.
- **Modern UI**: Clean and modern design.

## How to run
Simply open `index.html` in your web browser! No local server or build steps are required. 
You can test it with the provided `example.md` file.
